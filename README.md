# data-storytelling-crash-course
This is a crash course in data storytelling. Over three sessions of roughly an hour, I go through the basics of data ingestion and cleaning, modeling, and presentation. After the sessions (and some homework), the student should be able to produce a dashboard in Python.
## Instructions
Download the .ipynb notebook and .csv file. Open the notebook with VS Code (Codium), Google Colab, a Jupyter server, or however you choose to consume .ipynb files.
